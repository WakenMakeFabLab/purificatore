* 6 fogli (700x400mm) 3 mm multi-plex

  5x 3mmm LED bianco brillante

  5x resistente per LED
  Regolatore di tensione 12v in, 5v out
  12v power supply (1.5A+)
  MQ 135 gas and smoke sensor
  Prototyping board
  2 pin connectors
  5 pin connectors
  Wires
